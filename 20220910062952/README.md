# Copy and paste in tmux

1. Add ```setw -g mode-keys vi``` in your tmux config.
1. ```Ctrl + [``` to enter tmux vi copy mode.
1. Move around with vim navigation keys.
1. Begin highlight with ```Space```.
1. Copy with ```Enter```.
1. Paste with ```Ctrl + ]```
